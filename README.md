# K5Web

- 世界业余无线电日 K5Web 正式开源，添加开源许可协议。
- 感谢所有 HAM 。

## 简介

K5Web 用于对兼容业余无线电台 UV-K5 写频、更新固件、写入星历等。

## 功能列表

- 固件版本检测
- EEPROM 大小检测
- 信道管理
- 启动画面文字管理
- MDC 本地侧音控制（仅支持我的 LTS 固件）
- 备份/还原 EEPROM
- 固件升级
- 开机图片（LOSEHU 固件）
- 字库写入（LOSEHU 固件）
- 星历写入（LOSEHU 固件）
- DTMF ID 设置
- 收音机频道管理
- MDC 联系人管理（LOSEHU 固件）

## 开发
### 安装依赖
```
yarn
```
### 开发
```
yarn dev
```
### 编译
```
yarn build
```
